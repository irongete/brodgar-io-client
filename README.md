# Brodgar.io Client

[![Latest release](https://img.shields.io/github/v/release/irongete/brodgar-io-client)](https://github.com/irongete/brodgar-io-client/releases/latest) &nbsp;·&nbsp; **[Download](https://github.com/irongete/brodgar-io-client/releases/latest)**

A ready-to-run [Haven & Hearth](https://www.havenandhearth.com/) client that
showcases the **Brodgar.io** integrations — and a worked example of how to add
them to your own client.

## Integrations

| Integration | Library | Adds |
|---|---|---|
| Proximity Voice Chat | [brodgar-io-voice-client](https://github.com/irongete/brodgar-io-voice-client) | Positional voice: options panel, push-to-talk, per-player mute |

## Play

Download the zip from the [latest release](https://github.com/irongete/brodgar-io-client/releases/latest),
unzip it, and double-click `run.bat` (needs Java 11+).

## Build from source

`ant` produces a runnable `bin/` (double-click `bin/run.bat`). See the upstream
[`README`](README) for the Haven & Hearth client's technical details.
